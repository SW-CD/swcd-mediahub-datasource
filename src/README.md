## Overview

The MediaHub Data Source integrates **MediaHub OSS** with Grafana, allow for an easy to use integration of files and media (image, video, audio).

A typical use-case is the display of annotated camera images or accessing files generated by a backend service.

![MediaHub Plugin Overview](https://raw.githubusercontent.com/[YOUR-ORG]/[YOUR-REPO]/main/src/img/screenshot_overview.png)

### Key Features
* **Rich Media Proxying:** Securely stream images, audio, and video files directly into Grafana panels without exposing your MediaHub API to the public internet. Supports HTTP Range requests for native, buffered video scrubbing.
* **Dynamic Metadata Tables:** Flatten complex, dynamic MediaHub custom fields into structured Grafana DataFrames for easy sorting and filtering.
* **Dashboard Variables:** Fully supports Grafana template variables to dynamically swap databases or filter entries on the fly.

## Requirements

To use this plugin, you will need:
* **Grafana** v10.0.0 or higher.
* A running instance of **MediaHub OSS** v2.

## Getting Started

### 1. Configuration
1. Navigate to **Connections > Data sources > Add new data source** and select **MediaHub**.
2. Enter your MediaHub **Base URL** (e.g., `https://mediahub.internal.com`).
3. Provide your **Username** and **Password**.
4. Click **Save & test**. The plugin will automatically negotiate an access token and verify the connection.

### 2. Querying Data
The Query Editor offers three primary models:

* **Get Metadata Table:** Retrieves a paginated list of entries from a specific database. You can apply search filters and generate secure proxy links for media previews.
* **Get Preview / Entry:** Fetches a specific media file by its Entry ID. Perfect for pairing with dashboard variables to display a single selected file in a panel.
* **Get Audit Logs:** Only avialble if the **Username** belongs to an admin user. Retrieves the system audit trail, parsing the complex JSON event details into a clean, readable table format.

### 3. Displaying Media in Dashboards
To render dynamic media (Images, Video, Audio, or Download buttons) directly inside a dashboard, we highly recommend installing the free **Business Text Panel** (formerly Dynamic Text Panel) from the Grafana catalog.

Because the MediaHub plugin automatically exposes the `mime_type` of every file, you can use pure Handlebars templating to create a universal media player without executing any JavaScript or compromising Grafana's security settings.

Create a **Business Text Panel** with HTML content, set your query to "Get Entry", and paste the following snippet into the **Content** editor:

```html
<div style="display: flex; justify-content: center; align-items: center; height: 100%; width: 100%;">
  
  {{#if (contains mime_type "video/")}}
    <video src="{{entry}}" controls style="max-width: 100%; max-height: 100%; object-fit: contain; border-radius: 8px;"></video>
  
  {{else if (contains mime_type "audio/")}}
    <audio src="{{entry}}" controls style="width: 100%; max-width: 400px;"></audio>
  
  {{else if (contains mime_type "image/")}}
    <img src="{{entry}}" style="max-width: 100%; max-height: 100%; object-fit: contain; border-radius: 8px;" />
  
  {{else}}
    <a href="{{entry}}" target="_blank" style="padding: 10px 20px; background-color: #3274D9; color: white; border-radius: 4px; text-decoration: none; font-weight: bold; font-family: sans-serif;">
      Download File
    </a>
  {{/if}}

</div>

```
